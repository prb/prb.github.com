I've always liked the little differences that make every platform special.

For example, consider the following C++ program:

#include <iostream.h>
main() {
  int i;
  i=0;
  cout << ++i << i++ << ++i;
}

Compiling with g++ (version 3.3) on my PowerBook yields the output 113, but on a Linux box, compiling with g++ (version 2.95.4) outputs 311.

Here's a related brainteaser, albeit in Java. Consider the following Java fragment:

int x=0;
System.out.println(++x + "" + x++ + "" + ++x);
  1. What does it output?
  2. Can parentheses be inserted so that the output is different?
