After experiments with Microsoft Word and DocBook (both of which turned out to be poor choices but for different reasons), we're using the Confluence wiki for the documentation for PXE, and I've been enjoying TimTam instead of a browser-based client.

Named for the tasty cookie from Oz that can be used as a straw, TimTam is an Eclipse-based thick client (a "thiki", as I would call it) for Confluence with a useful interface -- left-hand tree for wiki pages and a normal-sized window with reasonable key bindings, content-assist for macros, and a source/preview dual view for editing content. TimTam's still pre-1.0, but the tip builds and works well on Mac OS X with Eclipse 3.1M5a.
